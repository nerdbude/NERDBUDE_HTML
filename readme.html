WAS IST DIE NERDBUDE
--------------------
Bei der Nerdbude handelt es sich um meinen wabernden, stetig wachsenden,
glühenden, klickenden und klackenden dump für Code, Hardware, Projekte und Gedöns.
Die Artikel sind meine persönlichen Meinungen, Projekte oder Erfahrungen und Anleitungen
zu Projekten anderer. Wenn etwas bei euch nicht funktioniert bin nicht ich Schuld.


KOMMENTARE
----------
Die Kommentarfunktion im eigentlichen Sinn gibt es hier nicht.
Das hat mehrere Gründe. Erstens will ich hier keine Daten sammeln. Zweitens ist die Nerdbude
in reinem HTML/CSS geschrieben, sprich kein PHP oder SQL das im Hintergrund läuft um eine
Kommentarfunktion zu ermöglichen und drittens soll das hier keine Troll-sammelstelle werden.

Da mir allerdings auch Fehler unterlaufen werden und ich ganz gerne dazulerne, wenn ich Fehler mache
gibt es die Möglichkeit auf Github zu jedem Artikel ein Issue auf zu machen. Schau ich mir dann an
und werde die Änderungen (gerne auch mit Namen oder Pseudonym) hier einbauen.
Hier liegen die ganzen Artikel auf Github:
https://www.github.com/nerdbude/nerdbude_html

WIE FUNKTIONIERT DIE NERDBUDE
-----------------------------
Die Nerdbude ist als Blog konzipiert. Die neusten Artikel werden als erstes angezeigt.
Auf die index.html kommt ihr immer über [$nb].

Auf technischer Seite steckt hinter der Nerdbude ein recht simples Gerüst gebaut in HTML5.
Farben, Links, Tabellen, Absätze etc. regelt die style.css
Das Syntax Highlighting übernimmt CSSYN. Ein kleiner CSS Eigenbau der Code-Syntax farblich hervorhebt
ohne Java Script benutzen zu müssen (CSSYN (Github))
Geschrieben werden die Beiträge, sowie Design und HTML im besten Texteditor der Welt - VIM!

PRIVATSPHÄRE
------------
Die Nerdbude setzt selber keine Cookies, benutzt keine Tracker oder ähnlichen spy-shit.
Die Verbindung ist per SSL abgesichert.
ABER! 100%ige Sicherheit gibt es nicht. Der Provider trackt mit Sicherheit IPs und loggt Zugriffe.
Ich arbeite daran die Nerdbude umzuziehen und selber zu hosten. Das dauert allerdings noch.
Wenn es soweit ist wird es einen Artikel dazu hier geben.
