cssyn



Um Code oder Linux Befehle auf der Website darzustellen braucht es einen Syntax-Highlighter. Die meisten sind allerdings nur in Verbindung mit JavaScript - will man nicht unbedingt haben.
Also das ganze mal als reine CSS Lösung versuchen. Das ist cssyn, ein leichtgewichtiger Syntax Highlighter in purem CSS. Aktuell zwar noch in der Alpha-Version aber das einfache highlighting funktioniert bereits. Zu erst einmal die Resultate:

Zur Installation:

Wer cssyn benutzen will kopiert sich entweder die cssyn.css in seinen /css folder auf dem Webserver und verlinkt sie in der *.html oder kopiert den Inhalt in seine eigene style.css. Damit ist cssyn schon einsatzbereit.

cssyn besteht aus dem "codeblock", der gibt die allgemeinen styles vor und den verschiedenen Klassen, welche die einzelnen Module im code wiedergeben und formatieren.

Der codeblock wird wie folgt definiert:

<div id="cssyn"></div>

Um den Code oder den Befehl einzufärben werden <span> tags benutz, z.B.:

<span class="comment"> /* simple comment */ </span>

Das gibt einen simplen Kommentar aus und benutzt die Formatierung für Kommentare.

Aktuell ist das cssyn.css file noch sehr übersichtlich. Ich werde es nach und nach weiter ausbauen mit mehr Colorschemes, Klassen, und angepasster Syntax an verschiedene Sprachen.
