iwtb - colorscheme





Wer endlich mal neue Farben im Terminal oder in Vim braucht, kann ab sofort gerne das IWTB colorset dafür nehmen. Eigens von mir zusammengeschraubt und für gut befunden. Ein farbenfrohes set für dunkle Umgebungen mit einem Hauch 80s/90s. Aktuell gibt es auf Github die Terminal Variante und die Beta für Vim. Wenn ihr noch andere Tools habt und das Theme dort verwenden wollt, steht es euch natürlich frei das ganze umzusetzen. Einzige Vorraussetzung ist die contribution eurer Umsetzung zu meinem Github Repo. :)
